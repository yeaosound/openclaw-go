## 📝 關於本項目

**OpenClaw Go** 是基於 [OpenClaw](https://github.com/openclaw/openclaw) 的優化分支：

- **Go** = **Global** + **Optimized**
  - 🌍 **Global**：全球化支持，多語言適配
  - ⚡ **Optimized**：性能優化，代碼結構優化
- 🇹🇼 **繁體中文支持** - 計劃中
- 🌏 **多語言支持** - 簡體中文、繁體中文、英語及更多語言
